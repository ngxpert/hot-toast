🔗 Built-in Themes (New in v7.0.0)

🎨 Pick your visual style

Hot Toast ships with 6 built-in themes. Import only what you need — the base styles give you toast and snackbar out of the box. The four new themes each have their own stylesheet to keep your bundle lean.

Themes can be set globally via provideHotToastConfig({ theme: 'material' }) or per-toast via the theme option.

⚙️ Global configuration

Set a theme once for your whole app. Individual toast calls can still override it with their own theme option.

🌙 Dark mode

All 4 new themes respond to @media (prefers-color-scheme: dark) automatically. For a manual toggle, add hot-toast-dark-theme to any parent element — it forces dark mode on every themed toast inside it, regardless of OS preference.
