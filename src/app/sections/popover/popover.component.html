The Popover API is a native browser feature that renders elements in the browser's top layer, ensuring they appear above all other content without z-index conflicts.

Hot Toast now uses the Popover API by default (when supported), providing better stacking context and avoiding z-index issues with dialogs, modals, and other overlay components.
